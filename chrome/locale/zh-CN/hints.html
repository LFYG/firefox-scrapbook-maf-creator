Saving Entire Document Trees

You can combine your entire Scrapbook document tree, or a sub-tree, into a single MAFF. Select "Output As HTML" from Scrapbook's "Tools" menu. Select the folder containing the documents you want to combine, check "Output with frame" and "When the process done, open the HTML file" then press start. Save the resulting page in Scrapbook and then create your MAFF from the saved page. You will now have a MAFF containing all of the selected pages and a convenient menu.

Combining Documents

Right-click a folder in Scrapbook's document tree and select "Combined View". Scrapbook will create a single page containing the folder's contents in series. Save this page in Scrapbook and then create your MAFF from the saved page. You will now have a single MAFF containing the serialised pages.

Alternatively, select "Combine Wizard" from Scrapbook's "Tools" menu. Drag the pages you want to combine into teh workspace, select "Next" and then "Combine". Scrapbook will save the new page. Create your MAFF from the saved page.
