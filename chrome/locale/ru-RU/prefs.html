Preferences

You can access the options dialogue from the add-on manager or from Scrapbook's "Tools" menu in the sidebar header, under the "Add-on Functions" sub-menu.

 

The options for the extension are simple.

 

Output path

The path to which the output files will be written. Help files will no longer be written to Scrapbook's data directory for the stored site by default. You are now prompted to configure a valid path.

Store multiple sites in one archive

If this option is set, when archiving a Scrapbook folder containing multiple stored sites all sites will be stored in a single MAF archive. When the archive is opened, each site will open in a separate tab.

If this option is not set, when archiving a Scrapbook folder containing multiple stored sites each site will be stored in it's own MAF archive. This is the same as archiving each site individually.

Help Button

Opens the inline help.
